304 Online

Production-ready web game implementation based on documented rules and feature requirements, with secure local room/session persistence and hardening for browser delivery.

Quick start guide (new players)
  1. Set your display name and choose the rule profile.
  2. Create a room, then share the invite link shown in the lobby.
  3. Other players join with the code using “Join room”.
  4. When all human players are marked ready, the host starts the match.
  5. Follow on-screen phase prompts to bid, choose trump, and play cards.

For classic 304, use the classic profile with 4-seat mode. Six-seat mode is used for the 304-36 variant.
